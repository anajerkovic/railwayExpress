# Web Programiranje - LV2

Ovaj projekt je jednostavna web aplikacija izrađena za kolegij Web Programiranje, LV2. Koristi Node.js, Express i EJS za prikazivanje galerije slika.

## Opis

Aplikacija sadrži:
- Početnu stranicu
- Galeriju slika sa lightbox efektom
- Grafikon stranicu

Slike se učitavaju iz `images.json` datoteke i prikazuju dinamički koristeći EJS templating.

## Instalacija

1. Instaliraj Node.js ako već nemaš.
2. Kloniraj ili preuzmi projekt.
3. U direktoriju projekta pokreni: `npm install`

## Pokretanje

Pokreni server sa: `npm start` ili `node server.js`
